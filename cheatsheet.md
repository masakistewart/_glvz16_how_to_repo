# Git Cheatsheet

### Start a new project

```shell

$ mkdir project_name
$ git init
$ touch readme.md
$ git add readme.md
$ git commit -m "initial commit"
```
### Do some work and then save it

First, do some work!


Make some changed, put them in the box, label the box.
```shell
$ git status
$ git add <whatever file>
git status
$ git commit -m " "
```
